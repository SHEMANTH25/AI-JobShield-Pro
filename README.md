# 🤖 AI JobShield Pro

An AI-powered Recruitment Fraud Detection Platform that helps job seekers identify fake job postings, phishing websites, suspicious recruitment messages, fraudulent PDFs, resumes, and images using Machine Learning and Explainable AI.

## 🚀 Features

- ✅ Job Description Fraud Detection
- ✅ URL Security Analysis
- ✅ Resume Scam Detection
- ✅ PDF Analysis
- ✅ Image OCR Analysis
- ✅ Explainable AI Predictions
- ✅ Fraud Score Calculation
- ✅ Dashboard Analytics
- ✅ Analysis History
- ✅ PDF Report Generation
- ✅ Responsive Modern UI

## 📂 Modules

### 📝 Job Analyzer
Analyze job descriptions and identify recruitment scams.

### 🌐 URL Analyzer
Detect phishing websites and suspicious recruitment URLs.

### 📄 PDF Analyzer
Analyze PDF job advertisements for fraudulent content.

### 🖼️ Image Analyzer
Extract text from recruitment images using OCR and analyze for scams.

### 📑 Resume Analyzer
Evaluate resumes against job requirements.

### 📊 Dashboard
Visualize fraud detection statistics and analytics.

### 📜 History
Store and review previous analyses.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/SHEMANTH25/AI-JobShield-Pro.git
```

### Backend

```bash
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
